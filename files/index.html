Microsoft Azure Serverless Deployment

Continuous Integration and Continuous (CI/CD) Deployment Project

  • Serverless deployment of website fully automated using Azure DevOps Organization
  • Deployment of a website without the need to provision and manage virtual machines
  • Provisioned pipelines to deploy the website fully automated in three different stages - Testing, Quality Assurance, and Production
  • Deployment of a website that is completely serverless using Azure Storage Account, Blobs (object storage), Static website
